The Electoral College - How does it work?

The electoral college determines the outcome of presidential elections, not the majority vote. In 2016, Hilary Clinton had more votes than Donald Trump, but Trump was still elected. So, how did this happen?

Each state has a certain amount of electoral votes based on the population of the state. Wyoming, the state with the lowest population, has 3 electoral votes and California, the state with the highest population, has 55 electoral votes.

A simple summary: If the majority of the population of a state votes for the democratic candidate, the whole state will turn blue and all of the electoral votes for that state will go towards the democratic candidate. In some states, the state color is based on the popular vote in each congressional distric. You need 270 of these electoral votes to win the election.
